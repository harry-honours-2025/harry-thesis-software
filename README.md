### Instance Construction

An **instance** is defined by the following facts:

- `node(X,T)`
- `vehicle(V,C)`
- `availability(V,B,E)`
- `arc(X,Y,V,C,D)`
- `month(M,B,E)`
- `demand(I,X,Y,M,Q,R,F,T)`
